# 🚀 Sistem Informasi & Pengelolaan Surat Desa

Sistem Informasi Desa berbasis Web yang dirancang untuk mempermudah masyarakat dalam mengajukan surat administrasi secara online, serta memudahkan perangkat desa dalam mengelola data dan menerbitkan surat resmi.

---

## ✨ Fitur Utama

- **Pengajuan Surat Online:** Memudahkan warga mengajukan berbagai jenis surat (SKU, SKTM, Surat Keterangan Domisili, Surat Keterangan Nikah, dll.).
- **Editor Format Surat (WYSIWYG):** Panel admin dilengkapi dengan template editor interaktif berbasis CKEditor untuk menyesuaikan format dan variabel surat secara fleksibel.
- **Pengelolaan Jenis Surat:** Admin dapat menambahkan, merubah, atau menghapus jenis surat pelayanan secara dinamis.
- **Cetak Surat Otomatis:** Generasi dokumen surat siap cetak langsung dari sistem.
- **Manajemen Berita & Informasi Desa:** Fitur untuk mempublikasikan artikel dan pengumuman desa.

---

## 🛠️ Teknologi yang Digunakan

- **Language:** PHP 8.x
- **Database:** MySQL / MariaDB
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Rich Text Editor:** CKEditor (WYSIWYG Editor Template Surat)
- **Database Management:** phpMyAdmin
